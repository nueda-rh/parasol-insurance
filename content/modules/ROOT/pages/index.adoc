== {ic} の概要

この {ic-lab} では、さまざまな AI/ML テクノロジを組み合わせて、ビジネス上の問題に対する価値あるソリューションを生み出す方法を説明します。

このラボに含まれる情報、コード、モデル、および様々なテクニックは、最初のプロトタイプをどのように組み立てるかを示したものです。これは、最終的なビジネス要件に対処するための決定的な方法ではありません。

==  免責事項

この {ic-lab} は、お客様が {rhoai}  を使用して構築できるものの例です。{rhoai}  には、保険金請求処理に関連する特定の機能はありません。

この {ic-lab} では、大規模言語モデル (LLM) と画像処理モデルを使用します。これらのモデルはオープンソースで提供されており、 {rhoai} 製品には含まれていません。
これらのモデルの品質は、プロトタイプには十分です。本番環境で使用する適切なモデルを選択することは、多くの実験と調整を必要とする複雑な作業です。この {ic-lab} では、この側面は取り上げません。
