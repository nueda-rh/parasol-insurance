== プロジェクトを開く

* まず、 {rhoai} ダッシュボードアプリケーションで、左側のメニューから「Data Science Projects」に移動します。
+
[.bordershadow]
image::02/02-02-ds-proj-nav.png[]

* 次に、 {user} というプロジェクトを開きます。

* プロジェクト内には、すでに作成済みのいくつかのアイテムが表示されているはずです。
+
[.bordershadow]
image::02/02-02-pre-created-components.png[]

* これらのコンポーネントは以下の通りです。
1. Workbenches - これは、実験やトレーニングを行うための環境です。
2. Cluster storage - これは、ワークベンチの永続的なストレージです。
3. Data connections - 既存の S3 ストレージとの通信に必要なすべての情報を含んでいます。モデルやパイプラインの成果物を保存するためにこれを使用します。
4. Pipelines - データサイエンスパイプラインをインポートしたり実行したりすることができます。
