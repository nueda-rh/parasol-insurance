本ワークショップでは事前に必要なプロジェクトを作成し、さまざまな成果物を追加しています。

// 製品についてさらに詳しく学びたい場合は、代わりに次の「DIY」セクションで、すべてをゼロから作成する手順を説明します。

== プロジェクトを開く

* まず、 {rhoai} ダッシュボードアプリケーションで、左側のメニューから「Data Science Projects」に移動します。
+
[.bordershadow]
image::02/02-02-ds-proj-nav.png[]

* 次に、 {user} というプロジェクトを開きます。

* プロジェクト内には、すでに作成済みのいくつかのアイテムが表示されているはずです。
+
[.bordershadow]
image::02/02-02-pre-created-components.png[]

* これらのコンポーネントは以下の通りです。
1. Workbenches - これは、実験やトレーニングを行うためのIDE環境です。
2. Cluster storage - これは、ワークベンチの永続ストレージ（Persistent Volume）です。
3. Data connections - 既存の オブジェクトストレージとの通信に必要なすべての情報を含んでいます。モデルやパイプラインの成果物を保存するためにこれを使用します。
4. Pipelines - データサイエンスパイプラインをインポートし実行します。
