include::_attributes.adoc[]

このトレーニング セッションでは、 {rhoai} がデプロイされた単一の {ocp} クラスターをプロビジョニングしました。

モデルは GPU ではなく CPU でホストされるため、この特定のラボは個人によるデモとして使用する必要があります。LLM は CPU では動作が遅く、LLM をクエリしようとするとこのデモでそのことがわかります。

== 環境情報

カスタマイズされたバージョンの手順を使用している場合は、以下の情報が正しく表示されます。そうでない場合は、代わりにプレースホルダー値が表示されます。

* アカウント ID: `{user}`
* パスワード: `{password}`

新しいウィンドウまたはタブで、次の URL を開いてログインします:

* 共有環境の Red Hat {rhoai} ダッシュボード URL:
** https://rhods-dashboard-redhat-ods-applications.{openshift_cluster_ingress_domain}/[https://rhods-dashboard-redhat-ods-applications.{openshift_cluster_ingress_domain}/,window=_blank]
* `Login with OpenShift` ボタンをクリックしてください
+
[.bordershadow]
image::02/02-01-login3.png[width=50%]
* 下に示されている通りあなたの資格情報を入力してください
* 結果はこのようになります:
+
[.bordershadow]
image::02/02-01-login1.png[width=50%]

これで {ic} を開始する準備ができました。
