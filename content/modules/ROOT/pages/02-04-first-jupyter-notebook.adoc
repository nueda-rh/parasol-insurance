= ノートブックで実行中のコード
include::_attributes.adoc[]

注：Jupyter Notebook にすでに慣れている場合は、次のセクションに進んでください。

ノートブックとは、書式付きテキストやコードを表示できるセルを持つ環境です。

これは空のセルです。

[.bordershadow]
image::02/02-05-cell.png[Jupyter Cell]

これはコードを含むセルです。

[.bordershadow]
image::02/02-05-cell_code.png[Jupyter Cell Code]

コードセルには、対話的に実行できるPythonコードが含まれています。コードを修正して実行することができます。コードは、お使いのコンピュータやブラウザ上ではなく、接続している環境で直接実行されます。

コードセルは、ノートブックインターフェースまたはキーボードから実行できます。

* *ユーザーインターフェースから:* セルを選択し（セル内またはセルの左側をクリック）、ツールバーの「実行」をクリックします。
+
[.bordershadow]
image::02/02-05-run_button.png[Jupyter Run]

* *キーボードから実行する場合:* `CTRL`+`ENTER` を押すとセルが実行され、`SHIFT`+`ENTER` を押すとセルが実行され、次のセルが自動的に選択されます。

セルを実行すると、そのコードの結果とセルが実行された日時に関する情報が表示されます。次の例を参照してください。

[.bordershadow]
image::02/02-05-cell_run.png[Jupyter run cell]

ノートブックを保存すると、コードと結果が保存されます。コードにアクセスできる状態で、プログラムを再度実行することなく、結果を確認するためにノートブックを再度開くことができます。

ノートブックという名称は、物理的な「ノートブック」のようなものだからです。コード自体に加えて、設定したパラメータも含め、実験に関するメモを取ることができます（実際に行う実験です）。実験の出力結果はインラインで表示でき（これはセルを実行した後の結果です）、取っておきたいメモもすべて表示できます（メニューからセルの種類を「コード」から「Markdown」に切り替えることで、そうすることができます）。
