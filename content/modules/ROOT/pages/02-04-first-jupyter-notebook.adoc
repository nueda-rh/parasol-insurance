= Jupyter Notebookでのコード実行
include::_attributes.adoc[]

注：Jupyter Notebook にすでに慣れている場合は、次のセクションに進んでください。

ノートブックとは、書式付きテキストやコードをセルとして表示、実行できるPythonのIDE環境です。

以下は空のセルです。

[.bordershadow]
image::02/02-05-cell.png[Jupyter Cell]

以下はコードを含むセルです。

[.bordershadow]
image::02/02-05-cell_code.png[Jupyter Cell Code]

コードセルには、インタラクティブに実行できるPythonコードが含まれています。この特定のセルの中でコードを修正して実行し、結果を確認することができます。コードは、お使いのコンピュータやブラウザ上ではなく、接続している環境で実行されます。

コードセルは、ノートブックインターフェースまたはキーボードから実行できます。

* *ユーザーインターフェースから実行する場合:* セルを選択し（セル内またはセルの左側をクリック）、ツールバーの「実行」をクリックします。
+
[.bordershadow]
image::02/02-05-run_button.png[Jupyter Run]

* *キーボードから実行する場合:* `CTRL`+`ENTER` もしくは、`SHIFT`+`ENTER` を押すとセルが実行され、次のセルが自動的に選択されます。

セルを実行すると、そのコードの結果が表示されます。次の例を参照してください。

[.bordershadow]
image::02/02-05-cell_run.png[Jupyter run cell]

ノートブックファイルを保存すると、コードと結果が保存されます。これによりプログラムを再度実行することなく、ノートブックを再度開くことだけで実行結果を確認できます。

ノートブックという名称は、物理的な「ノートブック」のようなものだからです。コード自体に加えて、設定したパラメータも含め、実験に関するメモを取ることができます。実験の出力結果はインラインで表示でき、取っておきたいメモもすべて表示できます（メニューからセルの種類を「コード」から「Markdown」に切り替えることができます）。

== 今すぐ試してみましょう

基本的なことを理解したので、実際に試してみましょう！

=== 手順

ワークベンチで：

. 左側のナビゲーションメニューで、次のフォルダに移動します：`parasol-insurance/lab-materials/02`
. `02-04-first-jupyter-notebook.ipynb` というノートブックを開きます。
. 例えば、既存のセルを実行したり、セルを追加したり、関数を作成したりして、実験してみましょう。
+
この環境はユーザーごとに独立しており、何かを壊したり他のユーザーに影響を与えたりするリスクはありません。この環境の分離も、 {rhoai} がもたらす大きな利点です。
. オプションとして、Python 3カーネルを使用してコードセルを実行する新しいノートブックを作成することもできます。
.. 新しいノートブックを作成するには、*File ->New ->Notebook* を選択するか、ランチャーウィンドウのノートブックセクションにある Python 3 をクリックします：
+
[.bordershadow]
image::02/02-05-new_notebook.png[alt text]


ノートブックで実行する言語やバージョンが異なるカーネルを、それぞれ使用することができます。
