= OpenShift Terminal
include::_attributes.adoc[]

アプリケーションをデプロイするには、 {ocp} Web ターミナルを使用します。+
これは、 {ocp-short} コンソールから {ocp-short} クラスターでコマンドを実行できる Web ベースのターミナルです。

== {ocp} コンソールを開く

- 次のリンクから {ocp-short} コンソールにアクセスします:
+
[.bordershadow]
image::05/01-openshift-console.png[]

- プロンプトが表示されたら、認証情報を入力します。

== Web ターミナルを開く

- {ocp-short} コンソールで、右上隅にある Web ターミナル アイコンをクリックします:
+
[.bordershadow]
image::05/web-term-1.png[]

- 次に、既存のプロジェクトを選択して [**開始**]/[**Start**] をクリックします:
+
[.bordershadow]
image::05/web-term-2.png[]

- 数秒後、ターミナル インターフェイスが開きます。
- ここで、次の手順でコマンドを実行します。
